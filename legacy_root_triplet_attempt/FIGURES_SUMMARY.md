## How to Edit

### Using Adobe Illustrator (Professional)
1. Open the `.pdf` or `.svg` file
2. All elements are editable:
   - Text: change fonts, sizes, colors, content
   - Shapes: resize, recolor, reposition
   - Colors: modify any fill or stroke color
3. Save and export as needed

### Using Inkscape (Free)
1. Download: https://inkscape.org/
2. Open the `.svg` file (recommended) or `.pdf`
3. Edit text, colors, shapes, positions
4. Export to PNG at any DPI for final submission

### Using PowerPoint/Keynote (Quick edits)
1. Import PDF as object
2. Ungroup to edit individual elements
3. Limited but useful for simple changes

### Using Other Software
- **CorelDRAW**: Open PDF/SVG directly
- **Affinity Designer**: Excellent PDF editing
- **Gravit Designer** (free): Web-based, opens SVG

## Recommended Workflow

1. **For manuscript submission**:
   - Use PNG files (300 DPI, ready to submit)
   - Keep PDF/SVG as master copies

2. **For revisions**:
   - Open PDF/SVG in editor
   - Make changes
   - Export new PNG at 300 DPI

3. **For presentations**:
   - Use PNG for slides
   - Or import SVG for infinite scaling

## File Sizes

- PNG: ~200-500 KB each (raster, 300 DPI)
- PDF: ~30-50 KB each (vector, compact)
- SVG: ~80-190 KB each (vector, with metadata)
